# Question 14 - Skip Invalid Temperature Readings

This project contains a small C++ console program that reads 10 temperature values from the user and ignores any invalid readings below 0°C. It calculates the sum and average only for valid temperature entries.

## Files

- `Question_14-Skip Invalid Temperature Readings Using.cpp` - main C++ source file.
- `Makefile.win` - Windows makefile generated by Dev-C++ for building the project.
- `Algorithm.txt` - brief pseudocode/algorithm description.

## Behavior

- Prompts the user to enter 10 temperature readings in Celsius.
- Treats any value below 0°C as invalid and skips it.
- Tracks the number of valid and invalid readings.
- Displays the sum and average of valid readings when at least one valid entry is provided.
- Prints a message if no valid readings were entered.
