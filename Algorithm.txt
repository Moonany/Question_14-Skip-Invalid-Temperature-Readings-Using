Start
Declare variables: temp (double), sum = 0.0 (double), validCount = 0 (int), average (double)
Use a for loop, i = 1 to 10:

Prompt user to enter temperature reading i
Read temp
If temp < 0:

Display "Invalid reading skipped."
continue (skip to next iteration immediately)


Add temp to sum
Increment validCount
Display "Valid reading accepted: [temp] °C"


End For
If validCount > 0:

average = sum / validCount
Display average of valid readings


Else:

Display "No valid readings entered."


Stop
